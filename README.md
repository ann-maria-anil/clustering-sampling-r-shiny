This repository contains an R Shiny application for designing a cluster sampling survey to estimate the population mean. The app allows users to select population size, number of clusters, allowable error, confidence level, and expected cost and time per cluster.

#### Features

Interactive cluster sampling design

Cluster information and sampling summary tables

Design of experiment visualization

Dynamic estimation of survey cost and time

#### Technologies Used

R, Shiny, ggplot2, dplyr, DT

#### How to Run

Open RStudio

Set working directory to this folder

Run:

shiny::runApp()
